# Grafana Dashboards

JSON-Definitions für Grafana — bei Setup importieren via Grafana-UI oder Provisioning.

## Empfohlene Dashboards

- **Response-Time-P95** pro Route (von Sentry-Performance)
- **Error-Rate** (Sentry-Issues über Zeit)
- **DB-Connection-Pool** (`pg_stat_activity`)
- **Memory / CPU** (Node-Exporter)
- **Disk** (für VPS)
- **Audit-Log-View** auf `audit_events`
