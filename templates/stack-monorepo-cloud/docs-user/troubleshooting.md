# Troubleshooting

## Login funktioniert nicht

1. E-Mail korrekt? (Tippfehler, alte Adresse)
2. Bestätigungs-Mail erhalten? (Spam-Ordner prüfen)
3. Browser-Cookies aktiviert?
4. Browser-Cache leeren

→ Wenn nichts hilft: support@__DOMAIN__

## Magic-Link kommt nicht an

1. Spam-Ordner prüfen
2. E-Mail-Adresse korrekt?
3. Bis zu 5min warten
4. Erneut anfordern (max 3/h)

## App lädt nicht / weiße Seite

1. Browser-Cache leeren
2. Anderer Browser ausprobieren
3. Status-Page prüfen: status.__DOMAIN__
4. Browser-Console (F12) auf Errors prüfen

## Daten verschwinden / sind weg

→ NIEMALS. Wenn ja: sofort support@__DOMAIN__. Wir prüfen Audit-Log und ggf. Backup.

## "Offline"-Banner erscheint

Du bist offline. Schreib-Aktionen werden lokal gespeichert und synchronisiert sobald wieder online.

## Performance ist langsam

- Aktuelle Browser-Version?
- Browser-Extensions (z.B. Werbeblocker können stören)
- Internet-Verbindung gut?
- Status-Page für Service-Issues
