# Erste Anmeldung

## Registrierung {#registration}

1. Klicke auf "Anmelden" oben rechts
2. Wähle "Konto erstellen"
3. Gib deine E-Mail + Passwort ein (min. 12 Zeichen, kein häufiges Passwort)
4. Klicke auf "Registrieren"
5. Du erhältst eine Bestätigungs-Mail
6. Klicke auf den Link in der Mail
7. Du bist eingeloggt!

## Anmeldung mit Magic-Link

Alternative ohne Passwort:
1. Gib deine E-Mail ein
2. Klicke auf "Magic-Link senden"
3. Öffne die Mail und klicke den Link
4. Du bist eingeloggt!

## Erstes Mal angemeldet — was nun?

- Profil ergänzen (Name, optional Avatar)
- Workspace erstellen oder einer Einladung folgen
- Erste Aktion durchführen
