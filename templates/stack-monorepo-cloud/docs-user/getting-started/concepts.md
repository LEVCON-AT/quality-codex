# Konzepte

## Workspace
Ein Workspace ist dein Arbeitsbereich. Du kannst mehrere haben (privat, beruflich, gemeinsame mit Familie).

## Rollen
- **Owner**: erstellt Workspace, kann alles
- **Admin**: verwaltet User, kann fast alles
- **Editor**: kann Inhalte anlegen+bearbeiten
- **Viewer**: kann nur lesen
- **Guest**: beschränkter Lesezugriff (eingeladen für einzelne Resource)

## Buttons {#buttons}
- **Primary**: Haupt-Aktion (blau)
- **Secondary**: weniger wichtige Aktion (grau)
- **Ghost**: tertiäre Aktion (transparent)

## I18n
Standard-Sprache: __DEFAULT_LOCALE__. Wechselbar in den Profil-Einstellungen.

## Dark-Mode
Folgt deiner System-Einstellung automatisch. Manuell wechselbar in den Einstellungen.

## Tastenkürzel
- `Tab`: zwischen Elementen wechseln
- `Enter`: Buttons aktivieren
- `Esc`: Modals/Drawer schließen
- `Cmd/Ctrl+K`: globale Suche
