# FAQ

## Wie ändere ich mein Passwort?

Profil → Sicherheit → Passwort ändern. Du musst dein altes Passwort kennen.

## Ich habe mein Passwort vergessen.

Login-Seite → "Passwort vergessen?" → E-Mail eingeben → Reset-Link kommt. Link 15min gültig.

## Kann ich meinen Account löschen?

Ja. Profil → Sicherheit → "Konto löschen". DSGVO-Art. 17-konform.

## Werden meine Daten verschlüsselt?

- TLS 1.3 für Übertragung
- Backups verschlüsselt (age) vor Upload
- Passwörter gehashed (argon2id)
- Sensitive Felder optional encrypted-at-rest

## Wer hat Zugriff auf meine Daten?

Du selbst + Workspace-Mitglieder gemäß Rolle. System-Admin nur bei Support-Anfragen mit Audit-Log.

## Ist die App offline-fähig?

Ja, Web-App ist PWA. Schreib-Aktionen werden lokal gepuffert und beim nächsten Online-Status synchronisiert.

## Welche Browser werden unterstützt?

Aktuelle 2 Versionen von Chrome, Edge, Firefox, Safari.

## Kann ich Daten exportieren?

Ja, Profil → "Daten herunterladen" liefert JSON mit allen deinen Daten (DSGVO Art. 20).
